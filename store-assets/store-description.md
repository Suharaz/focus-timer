# Focus Timer - Chrome Web Store Listing

## Short Description (132 chars max)
Track time spent on websites, set daily limits, build better habits with reminders, and stay focused with a beautiful fullscreen clock.

## Detailed Description (English)

**Focus Timer** helps you understand and control your browsing habits. Track how much time you spend on each website, set limits to stay productive, and build healthy digital habits.

### Key Features

**Time Tracking**
- Automatically tracks time spent on every website
- Real-time floating timer on each page
- Daily and all-time statistics
- Visual charts showing your browsing patterns

**Website Limits**
- Set daily time limits for distracting sites
- Get warned when approaching your limit
- Full-screen block when limit is exceeded
- Stay accountable to your goals

**Smart Reminders**
- Create custom reminders for healthy habits
- Interval-based (every X minutes) or daily scheduled
- Beautiful popup notifications you can't ignore
- Track habit completion with heatmap visualization

**Focus Clock**
- Fullscreen minimalist clock for deep work sessions
- Right-click extension icon → Focus Clock
- Perfect for Pomodoro technique or focused work

**Dashboard & Analytics**
- Beautiful dashboard with time charts
- Week/Month/All-time views
- Track visits and time per domain
- Habit tracking heatmap

**Cloud Sync**
- Sync data across devices with Google Drive
- Automatic daily backups
- Never lose your progress

**Privacy First**
- All data stored locally on your device
- Optional Google Drive sync (your own account)
- Open source - verify the code yourself
- No ads, no tracking, no data selling

### Perfect For
- Students who need to focus on studying
- Remote workers managing distractions
- Anyone building better digital habits
- Productivity enthusiasts

## Mô tả chi tiết (Tiếng Việt)

**Focus Timer** giúp bạn hiểu và kiểm soát thói quen lướt web. Theo dõi thời gian trên từng trang web, đặt giới hạn để tăng năng suất, và xây dựng thói quen số lành mạnh.

### Tính năng chính

**Theo dõi thời gian**
- Tự động theo dõi thời gian trên mọi trang web
- Đồng hồ nổi realtime trên mỗi trang
- Thống kê theo ngày và tổng cộng
- Biểu đồ trực quan về thói quen lướt web

**Giới hạn trang web**
- Đặt giới hạn thời gian hàng ngày cho các trang gây xao nhãng
- Cảnh báo khi gần đến giới hạn
- Chặn toàn màn hình khi vượt giới hạn
- Giữ bạn tuân thủ mục tiêu

**Nhắc nhở thông minh**
- Tạo nhắc nhở tùy chỉnh cho thói quen lành mạnh
- Theo khoảng thời gian (mỗi X phút) hoặc theo lịch hàng ngày
- Popup thông báo đẹp mắt, không thể bỏ qua
- Theo dõi hoàn thành thói quen bằng heatmap

**Đồng hồ Focus**
- Đồng hồ fullscreen tối giản cho phiên làm việc sâu
- Click phải icon extension → Focus Clock
- Hoàn hảo cho Pomodoro hoặc làm việc tập trung

**Dashboard & Phân tích**
- Dashboard đẹp với biểu đồ thời gian
- Xem theo Tuần/Tháng/Tất cả
- Theo dõi lượt truy cập và thời gian theo domain
- Heatmap theo dõi thói quen

**Đồng bộ Cloud**
- Đồng bộ dữ liệu qua Google Drive
- Backup tự động hàng ngày
- Không bao giờ mất dữ liệu

**Quyền riêng tư**
- Mọi dữ liệu lưu trữ cục bộ trên thiết bị
- Tùy chọn đồng bộ Google Drive (tài khoản của bạn)
- Mã nguồn mở - tự kiểm tra code
- Không quảng cáo, không theo dõi, không bán dữ liệu
